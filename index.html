Woven Data

Woven Data is a handmade coat, assembled from woven squares, each of which represents a tracked data point about me. Each square is tied to a story, documented in a book and website, that aims to highlight the dichotomy between how my data is self-representative through my eyes versus the eyes of those who profit from it.
